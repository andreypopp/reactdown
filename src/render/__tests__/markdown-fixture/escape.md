These should all get escaped:

Backslash: \\

Backtick: \`

Asterisk: \*

Underscore: \_

Left brace: \{

Right brace: \}

Left bracket: \[

Right bracket: \]

Left paren: \(

Right paren: \)

Greater-than: \>

Hash: \#

Period: \.

Bang: \!

Plus: \+

Minus: \-

**GFM:**

Pipe: \|

Tilde: \~

**Commonmark:**

Quote: \"

Dollar: \$

Percentage: \%

Ampersand: \&

Single quote: \'

Comma: \,

Forward slash: \/

Colon: \:

Semicolon: \;

Less-than: \<

Equals: \=

Question mark: \?

At-sign: \@

Caret: \^

New line: \
only works in paragraphs.

Two spaces:  
only works in paragraphs.
